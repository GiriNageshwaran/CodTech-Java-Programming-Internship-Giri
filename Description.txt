Description:
		This project is a console-based Java application that demonstrates file handling operations. 
		The program allows users to write, read, modify, and delete text files. 
		It also provides file statistics such as total lines, words, and characters.
		The application includes input validation and exception handling for safe execution.

Features:
		1. Write to File – Appends user input to the file.
		2. Read File – Displays file content.
		3. Modify File – Replaces specific words in the file.
		4. Delete File – Deletes file with confirmation.
		5. File Statistics – Shows total lines, words, and characters.
		6. Input Validation – Prevents invalid menu inputs.


Technologies Used:

		1.Java
		2.File
              	3.FileReader
              	4.FileWriter
		5.Scanner
		6.StringBuilder
	 	7.Exception Handling (try-catch)

How to Run:

		How to Run:
